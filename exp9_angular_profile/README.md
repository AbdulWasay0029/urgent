# Exp 9 - Angular Profile with REST API
## How to Run
1. Follow all steps from Exp 8 first
2. Replace profile component files with the code in `src/app/profile/`
3. Run:
```
ng serve
```
4. Open browser at `http://localhost:4200`
5. Click on "Profile" to see user data fetched from JSONPlaceholder API

### Troubleshooting
- If profile doesn't load, press `Ctrl + F5` to hard reload
- Or stop server (`Ctrl + C`) and run `ng serve` again
- Or delete `.angular` cache folder and run `ng serve`
