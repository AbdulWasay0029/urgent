# Exp 8 - Angular SPA with Menu Items
## How to Run
1. Open terminal and run:
```
ng new spa
cd spa
npm install -g @angular/cli
```
2. Generate components:
```
ng g c header
ng g c home
ng g c profile
```
3. Replace the generated files with the code in `src/` folder
4. Run:
```
ng serve
```
5. Open browser at `http://localhost:4200`
